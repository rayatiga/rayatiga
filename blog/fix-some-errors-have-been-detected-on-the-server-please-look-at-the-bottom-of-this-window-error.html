Error message show up while accessing some phpMyAdmin and appering the error log in bottom of the page.

Note: Warning, this is masking the problem!

Error Information

In my case, I want to export databases in localhost phpMyAdmin with xampp, but there error window say:

"Some errors have been detected on the server! Please look at the bottom of this window."
How to Fix

Add this line code inside xampp/phpMyAdmin/config.inc.php before ?> tag
or inside /etc/phpmyadmin/config.inc.php (outside using xampp):

$cfg['SendErrorReports'] = 'never';
After Fixing

Now, the windows say “Some errors have been detected on the server! Please look at the bottom of this window” should be gone.

And that's it. Problem should be solved!
