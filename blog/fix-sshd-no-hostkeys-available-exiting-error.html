Error message show up while trying to start up SSH service in Ubuntu WSL.

Error Information

Today while trying to start ssh server on my WSL Ubuntu installation, I got this error:

sshd: no hostkeys available -- exiting
How to Fix

After searching on the web, I found that the solution was really simple, at least in this case.
I only needed to run: 

ssh-keygen -A
After Fixing

If needed, you have to restart the SSH service. Run this command:

sudo systemctl restart ssh

And that's it. Problem should be solved!
