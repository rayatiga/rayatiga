[Article Title]

Error message show up while trying to close window XAMPP Control Panel.

Error Information

When I opened Adobe XD, I encountered an error like this:

"Exception EAccessViolation in module xampp-control.exe at 0025B2AE."
"Access violation at address 0065B2AE in module 'xampp-control.exe'. Read of address 00000042."

Is an error when pressing 'Exit' in the XAMPP Control Panel software. This error appears and is usually followed by not responding.
The reason is that the XAMPP Control Panel program is not running in Administrator.

How to Fix

Make xampp-control.exe running as an administrator.

  1. Go to directory: xampp/
  2. Scroll down and find xampp-control.exe
  3. Right click on that .exe
  4. And then, Properties
  5. In Properties go to tab Compability
  6. Checklist the Run this program as administrator
  7. Apply, and OK
After Fixing

After everything is done, let's try retesting again.
Run the xampp control panel and exit, the error window will not appear again.

And that's it. Problem should be solved!
